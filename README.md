## 📌 Descripción

**Control PC Flask** es una aplicación web que te permite manejar tu computadora desde el navegador de tu celular, siempre que ambos estén conectados a la misma red WiFi.

Ideal para:
- controlar música 🎵  
- manejar presentaciones 🎤  
- usar la PC a distancia 🖥️  

---

## ✨ Funcionalidades

### 🎵 Control Multimedia
- Play / Pause  
- Siguiente canción  
- Canción anterior  

---

### 🔊 Control de Volumen
- Subir volumen  
- Bajar volumen  
- Silenciar  

---

### 🖱️ Control del Mouse
- Movimiento del cursor  
- Click izquierdo  
- Click derecho  
- Scroll  

---

### ⚙️ Sistema
- 🔒 Bloquear PC  
- ⏻ Apagar sistema  

---

### 📱 Acceso Web
- Funciona desde cualquier celular  
- Solo necesitas el navegador  
- No requiere app  

---

## 🧠 ¿Cómo funciona?

```
Celular (Navegador)
        ↓
    Flask Server (PC)
        ↓
Control del sistema (mouse, teclado, volumen)
```

## 🛠️ Tecnologías

- Python 3

- HTML / CSS / JavaScript

## 📦 Instalación

```bash
git clone https://github.com/michaelGuerraQ/control-pc-flask.git
cd control-pc-flask
pip install -r requirements.txt
```

---

## ▶️ Ejecución

```bash
python app.py
```

---

## 📱 Uso

1. Ejecuta el servidor en tu PC  
2. Obtén tu IP local (ejemplo: `192.168.1.10:5000`)  
3. Abre esa dirección en tu celular  
4. ¡Controla tu PC desde el móvil! 🚀
