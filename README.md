Aplicación web desarrollada con **Python + Flask** que permite controlar tu computadora desde el celular dentro de la misma red WiFi.

---

## ✨ Funcionalidades

- 🎵 Control multimedia (Play / Pause, Next, Prev)
- 🔊 Control de volumen (Subir, bajar, mute)
- 🖱️ Movimiento del mouse
- 🖱️ Click izquierdo y derecho
- 🔄 Scroll
- 🔒 Bloquear PC
- ⏻ Apagar el sistema
- 📱 Acceso desde el navegador del celular

---

## 🛠️ Tecnologías

- Python
- Flask
- HTML / CSS / JavaScript
- ctypes (interacción con el sistema)
- sockets (detección de IP local)
